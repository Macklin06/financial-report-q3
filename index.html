import numpy as np

def calculate_var(portfolio_value, volatility, confidence=1.65):
    """
    Calculate Parametric VaR (95% Confidence)
    """
    return portfolio_value * volatility * confidence

Options Pricing Formula

Valuation based on the Black-Scholes model:

\[ C(S, t) = N(d_1)S - N(d_2)Ke^{-r(T-t)} \]

Where the volatility factor ($d_1$) is:

\[ d_1 = \frac{\ln(S/K) + (r + \frac{\sigma^2}{2})(T-t)}{\sigma\sqrt{T-t}} \]
