Risk Analysis Model

We implemented a new VaR (Value at Risk) calculation engine:


import numpy as np

def calculate_var(portfolio_value, volatility, confidence=1.65):
    """
    Calculate Parametric VaR (95% Confidence)
    """
    return portfolio_value * volatility * confidence

# Production Implementation
portfolio_total = 5000000  # $5M Portfolio
daily_volatility = 0.012   # 1.2%
risk_exposure = calculate_var(portfolio_total, daily_volatility)

print(f"Daily Risk Exposure: ${risk_exposure:,.2f}")

Options Pricing Formula

Valuation based on the Black-Scholes model:

\[ C(S, t) = N(d_1)S - N(d_2)Ke^{-r(T-t)} \]

Where the volatility factor ($d_1$) is:

\[ d_1 = \frac{\ln(S/K) + (r + \frac{\sigma^2}{2})(T-t)}{\sigma\sqrt{T-t}} \]
Application: Used for pricing 5,000+ option contracts this quarter
We use this formula to price our European call options. S is current stock price, K is strike price.
